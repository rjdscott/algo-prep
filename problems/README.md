# Problems

This directory contains coding interview problems organized by patterns/categories, following the Neetcode roadmap structure.

## Categories

### Core Patterns (Start Here)
1. **arrays_and_hashing**: Array manipulation, hash tables, frequency counting
2. **two_pointers**: Two pointer technique, palindromes, sorted arrays
3. **sliding_window**: Fixed/variable window problems, substrings
4. **stack**: Stack operations, monotonic stack, parentheses
5. **binary_search**: Search algorithms, rotated arrays

### Data Structures
6. **linked_list**: Singly/doubly linked lists, fast/slow pointers
7. **trees**: Binary trees, BST, DFS, BFS, traversals
8. **tries**: Prefix trees, word search
9. **heap_priority_queue**: Min/max heaps, K-th problems

### Advanced Patterns
10. **backtracking**: Combinations, permutations, subsets
11. **graphs**: Graph traversal, DFS, BFS, topological sort
12. **advanced_graphs**: Dijkstra, Prim's, union find
13. **dynamic_programming_1d**: Fibonacci-style, house robber patterns
14. **dynamic_programming_2d**: Grid problems, LCS, knapsack
15. **greedy**: Greedy choice, intervals, scheduling
16. **intervals**: Merging, overlapping intervals
17. **math_and_geometry**: Mathematical problems, matrix operations
18. **bit_manipulation**: Bitwise operations, XOR tricks

## File Format

All problems are Jupyter notebooks (.ipynb) with:
- Detailed markdown explanations
- Multiple solution approaches
- Complexity analysis
- Interactive test cases
- Visual explanations where applicable

## File Naming Convention

Files are named using the pattern: `{leetcode_number}_{problem_name}.ipynb`

Examples:
- `001_two_sum.ipynb`
- `217_contains_duplicate.ipynb`
- `049_group_anagrams.ipynb`

## How to Use

1. Start with easier categories (arrays_and_hashing, two_pointers)
2. Work through problems in order of difficulty
3. Try to solve each problem yourself before looking at solutions
4. Study multiple approaches and understand trade-offs
5. Run the test cases interactively to verify understanding

## Running Notebooks

```bash
# Install Jupyter
pip install jupyter notebook

# Launch Jupyter
jupyter notebook

# Or use JupyterLab
pip install jupyterlab
jupyter lab
```
